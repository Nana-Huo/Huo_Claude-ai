### 1. 推送代码到GitHub

#### 步骤1：确保Git已安装

```bash
# 检查Git版本
git --version
```

#### 步骤2：进入项目目录


#### 步骤3：配置GitHub远程仓库

```bash

# 添加HTTPS远程仓库

```

#### 步骤4：推送代码

```bash

git push -u origin master
```

**注意**：如果推送时需要身份验证，请使用您的GitHub个人访问令牌（PAT）作为密码。

### 2. 部署到Vercel

#### 步骤1：登录Vercel

- 访问 [Vercel官网](https://vercel.com)
- 使用GitHub账号登录

#### 步骤2：导入GitHub仓库

1. 点击"Add New" → "Project"
2. 搜索并选择 `Nana-Huo/dance-booking-app` 仓库
3. 点击"Import"

#### 步骤3：配置前端部署

- **Root Directory**: `client`
- **Framework Preset**: 选择 `Vite`
- **Build Command**: `npm run build`
- **Output Directory**: `dist`

#### 步骤4：添加环境变量

- **`VITE_API_BASE_URL`**: 后端API地址（例如：`https://your-backend-api.com`）

#### 步骤5：部署

点击"Deploy"按钮，Vercel将自动构建并部署前端应用。

### 3. 后端部署（可选）

后端代码可以部署到以下平台：

#### 选项1：Vercel Serverless Functions

1. 在项目根目录创建 `vercel.json` 文件：

```json
{
  "version": 2,
  "builds": [
    {
      "src": "client/package.json",
      "use": "@vercel/static-build",
      "config": {
        "distDir": "client/dist"
      }
    },
    {
      "src": "server/src/index.ts",
      "use": "@vercel/node"
    }
  ],
  "routes": [
    {
      "src": "/api/(.*)",
      "dest": "server/src/index.ts"
    },
    {
      "src": "/(.*)",
      "dest": "client/dist/$1"
    }
  ]
}
```

2. 重新部署项目到Vercel

#### 选项2：Render.com

1. 访问 [Render官网](https://render.com)
2. 登录并创建新的Web Service
3. 连接GitHub仓库
4. 配置构建和启动命令
5. 部署应用
