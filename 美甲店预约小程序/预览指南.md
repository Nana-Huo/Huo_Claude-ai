# 美甲店预约小程序 - 预览指南

## 📱 预览步骤

### 1. 使用微信开发者工具预览

1. **打开微信开发者工具**
   - 下载并安装微信开发者工具
   - 选择"小程序"项目类型

2. **导入项目**
   - 点击"导入项目"
   - 选择项目目录：`美甲店预约小程序`
   - 填写 AppID（测试可使用测试号）
   - 项目名称：美甲店预约小程序

3. **编译运行**
   - 点击"编译"按钮
   - 在模拟器中查看效果
   - 可以选择不同设备型号测试

### 2. 真机预览

1. **扫码预览**
   - 在开发者工具中点击"预览"
   - 用微信扫描二维码
   - 在手机上查看实际效果

2. **调试功能**
   - 点击"真机调试"
   - 可以在手机上实时调试
   - 查看控制台输出

## 🛠 实时修改支持

### 常见修改需求

1. **颜色主题调整**
```javascript
// app.wxss 中修改主色调
:root {
  --primary-color: #ff6b9d;  // 修改这里即可
}
```

2. **服务项目更新**
```javascript
// app.js 中修改 services 数组
services: [
  {
    id: 1,
    name: '新增服务',
    price: 199,
    // ...
  }
]
```

3. **美甲师信息调整**
```javascript
// app.js 中修改 staffList 数组
staffList: [
  {
    id: 1,
    name: '新美甲师',
    rating: 4.9,
    // ...
  }
]
```

### 快速修改方法

1. **文字内容修改**
   - 直接在对应的 .wxml 文件中修改文字
   - 或在 .js 文件的 data 中修改

2. **样式调整**
   - 在对应的 .wxss 文件中修改样式
   - 全局样式在 app.wxss 中修改

3. **功能逻辑调整**
   - 在对应的 .js 文件中修改逻辑
   - 数据模型在 models/ 目录中修改

### 实时预览修改

1. **保存文件**
   - 修改后保存文件（Ctrl+S）
   - 开发者工具会自动重新编译

2. **查看效果**
   - 在模拟器中查看修改效果
   - 刷新页面查看更新

3. **真机同步**
   - 真机调试模式下会实时同步
   - 或重新扫码预览
