## 🚀 如何开始使用

### 第一步：初始化配置
如果您是第一次使用，建议先进行完整初始化：

在菜单中输入 `1` 并按回车，然后按照提示完成配置：
- 选择 API 类型（推荐 api_key）
- 输入您的 API 密钥
- 选择 AI 模型（如 claude-sonnet-4-5）
- 配置语言偏好

### 第二步：使用代码生成功能
配置完成后，您可以：

1. **通过菜单选择**：在主菜单中选择相应的代码生成选项
2. **直接命令**：
   ```bash
   .\start_zcf.bat code
   ```

## 💡 提示

- 如果您需要重新配置系统，随时可以运行 `.\start_zcf.bat init`
- 如果遇到问题，检查 API 密钥是否正确，网络连接是否正常
- 定期运行 `.\start_zcf.bat check-updates` 保持系统更新
