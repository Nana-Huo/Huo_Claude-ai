# 修正导航指南

## 问题分析

您在尝试使用 `cd` 命令导航到项目目录时遇到了以下错误：

```
Set-Location : 找不到接受实际参数"code"的位置形式参数。
所在位置 行:1 字符: 1
+ cd C:\Users\霍冠华\Documents\trae_projects\claude code
+ ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    + CategoryInfo          : InvalidArgument: (:) [Set-Location]，ParameterBindingEx
    ception
    + FullyQualifiedErrorId : PositionalParameterNotFound,Microsoft.PowerShell.Comma
   nds.SetLocationCommand
```

**错误原因**：
PowerShell 中，当路径包含空格时，需要将整个路径用引号括起来。在您的命令中，`claude code` 之间的空格导致 PowerShell 将 `code` 视为一个单独的参数，而不是路径的一部分。

## 正确的导航命令

### 方法1：使用双引号
```powershell
cd "C:\Users\霍冠华\Documents\trae_projects\claude code"
```

### 方法2：使用单引号
```powershell
cd 'C:\Users\霍冠华\Documents\trae_projects\claude code'
```

### 方法3：使用命令提示符 (cmd.exe)
如果您更习惯使用命令提示符，可以：
1. 按下 `Win + R` 键
2. 输入 `cmd` 并按回车
3. 在命令提示符中，空格不需要引号（但使用引号也可以）：
   ```cmd
   cd C:\Users\霍冠华\Documents\trae_projects\claude code
   ```

## 验证导航是否成功

成功导航后，您可以使用 `dir` 命令查看目录内容：
```powershell
dir
```

您应该能看到以下文件：
- AI_Usage_Guide.md
- Claude使用入门.md
- start_zcf.bat
- 快速开始指南.md
- 等其他项目文件


## 其他有用的PowerShell技巧

### 1. 使用Tab键自动补全
在输入路径时，可以按 `Tab` 键自动补全文件夹名称，这有助于避免空格和拼写错误。

### 2. 查看当前目录
```powershell
pwd
```

### 3. 列出目录内容（详细）
```powershell
dir -la
```

### 4. 返回上一级目录
```powershell
cd ..
```
