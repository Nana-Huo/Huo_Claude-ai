# 快速开始指南

## 如何使用您的AI系统

### 第一步：初始化配置
首次使用需要进行初始化配置：
```bash
.\start_zcf.bat init
```

初始化过程会引导您：
- 选择API类型（推荐使用api_key）
- 输入API密钥
- 选择AI模型（如claude-sonnet-4-5）
- 配置语言偏好

### 第二步：启动AI
配置完成后，您可以通过三种方式使用AI：

#### 1. 交互式菜单
```bash
.\start_zcf.bat
```

这会显示一个交互式菜单，您可以通过键盘选择功能：
- 代码生成
- 代码分析
- 文档生成
- 更多功能

#### 2. 直接命令
使用特定命令执行任务：
```bash
# 生成代码
.\start_zcf.bat code

# 分析代码
.\start_zcf.bat analyze

# 检查用量
.\start_zcf.bat ccu
```

#### 3. 在VS Code中使用
如果您在VS Code中安装了Claude Code扩展：
1. 打开VS Code
2. 右键点击代码或文件
3. 选择"Claude Code"相关选项
4. 输入您的需求

### 第三步：使用示例

**示例1：生成一个JavaScript函数**
1. 运行`.\start_zcf.bat`
2. 选择"代码生成"选项
3. 输入需求："生成一个计算数组平均值的JavaScript函数"
4. AI会返回完整的函数代码

**示例2：分析代码**
1. 运行`.\start_zcf.bat analyze`
2. 输入或粘贴您的代码
3. AI会分析代码并提供改进建议

### 常用命令速查表

| 命令 | 功能 |
|------|------|
| `.\start_zcf.bat init` | 初始化配置 |
| `.\start_zcf.bat` | 交互式菜单 |
| `.\start_zcf.bat update` | 更新配置 |
| `.\start_zcf.bat ccr` | 配置模型代理 |
| `.\start_zcf.bat ccu` | 查看用量统计 |
| `.\start_zcf.bat check-updates` | 更新系统 |

### 故障排除

如果遇到问题：
1. 确保Node.js和npm已正确安装
2. 检查网络连接
3. 验证API密钥是否有效
4. 尝试重新初始化：`.\start_zcf.bat uninstall` 然后 `.\start_zcf.bat init`
